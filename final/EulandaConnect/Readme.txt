EulandaConnect is the PowerShell module collection to access features of EULANDA ERP. 
All functions support PowerShell help with 'Get-Help'. 
For example:

      PS C:> Get-Help Get-DesktopDir

This will give you information about the function, its parameters, and specifics.

Help is also available online at:
