# Out-Beep

## SYNOPSIS
Sends a beep to the speaker

## SYNTAX

```
Out-Beep [<CommonParameters>]
```

## DESCRIPTION
Sends a short sound to the speaker of the Windows PC.

## EXAMPLES

### Example 1:Sends a sound to the speaker
```powershell
PS C:\> Out-Beep
```

Sends a sound to the speaker.
